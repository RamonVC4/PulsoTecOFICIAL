# **PulsoTec**

PulsoTec es una plataforma web desarrollada para el uso interno del Instituto Tecnológico de la Laguna, orientada a la gestión integral del proceso de evaluación de investigaciones destinadas a la revista científica PulsoTec del TecNM.

## **Función**
La plataforma permite administrar de manera centralizada todas las etapas del proceso editorial, incluyendo:

- Carga y gestión de documentos por parte de los autores
- Asignación de revisores
- Revisión de trabajos directamente desde la plataforma
- Evaluación de la primera y, cuando aplique, segunda entrega
- Comunicación y entrega de resultados a los autores

Todo el proceso se realiza bajo un esquema de anonimato, garantizando la confidencialidad de los autores y promoviendo evaluaciones objetivas y libres de sesgos.

PulsoTec busca optimizar el flujo editorial, mejorar la transparencia del proceso de revisión y facilitar la colaboración entre autores, revisores y el comité editorial.
